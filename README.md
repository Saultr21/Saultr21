## Proyectos destacados

### 🔒 [pdf-privacy-tool](https://github.com/Saultr21/pdf-privacy-tool)
Herramienta web local para la **redacción de información sensible en PDFs**. Detecta y elimina PII de forma permanente (no solo visualmente) usando PyMuPDF y OCR con Tesseract. Stack: FastAPI + React/Vite + TypeScript.

`Python` `FastAPI` `TypeScript` `React` `OCR` `PyMuPDF`

---

### 🔬 [Clasificacion-de-imagenes-hispatologicas](https://github.com/Saultr21/Clasificacion-de-imagenes-hispatologicas)
Clasificador de **imágenes histopatológicas** mediante deep learning. Exploración de arquitecturas de visión por computador aplicadas a diagnóstico médico asistido.

`Jupyter Notebook` `Deep Learning` `Computer Vision` `Medical AI`

---

### 🤖 [redmine-ai-assistant](https://github.com/Saultr21/redmine-ai-assistant)
Asistente de IA integrado con **Redmine** para automatizar la gestión de tareas y proyectos. Implementa el protocolo MCP (Model Context Protocol) para interacción natural con la plataforma.

`Python` `MCP` `Redmine API` `Claude API`
